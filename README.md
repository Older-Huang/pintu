# jigsaw

智能拼图小游戏

## api

|参数   |说明    |类型                       |默认  |
|-------|-------|---------------------------|------|
|img   |图片路径 |string / SVGAElement / URL |-     |
|row   |行      |number                     |3     |
|col   |列      |number                     |3     |
|option|配置，传img、row、col或option二选一，option优先级较低|object|-|
